RAG (Retrieval-Augmented Generation) Page
Document Management for Tone of Voice
•	Document Import:
o	Easy drag-and-drop file upload interface supporting PDF, Word (.docx), and Text files.
o	Clear prompts guiding users on accepted file formats and size limits.
•	Project-Based Document Organization:
o	Documents grouped by associated projects.
o	Expandable sections for easy navigation and management.
•	Document Management Tools:
o	Add new documents directly under project categories.
o	Select multiple documents for batch operations (delete or archive).
o	Clear visual indicators for document status (processed, pending).
•	Search and Filter:
o	Quick search bar for instant document retrieval.
o	Filters to sort documents by project, status, or upload date.
